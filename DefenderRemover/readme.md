## 📋 Overview

**Defender Remover** is a PowerShell-based GUI tool that removes Windows Defender components from your system through an easy-to-use interface.

## ✨ Features

- 🖥️ **User-Friendly GUI** - Clean black-themed interface with ASCII art
- 🚀 **One-Click Operation** - Single button to execute removal process
- 🔧 **Automated Process** - Downloads required tools automatically
- 👑 **SYSTEM Privileges** - Executes commands with highest privileges
- 🧹 **Complete Removal** - Removes all Windows Defender directories and services
- 📊 **Real-time Status** - Visual feedback during each step
- 📊 **outputs** - log feedback for each step saved to a log in current directory

## 📦 Requirements

- Windows OS (10/11)
- PowerShell 5.1 or higher
- Administrator privileges
- Internet connection (for initial PsTools download)

## 🚀 Installation & Usage

### Step-by-Step Guide

#### Step 1: Disable Windows Antivirus Protection
Open **Windows Settings** → **Privacy & Security** → **Windows Security** → **Virus & threat protection** and disable all protection settings:
- Real-time protection
- Cloud-delivered protection
- Automatic sample submission
- Tamper Protection
- download Defender Remover

#### Step 2: Disable Windows Defender using DControl
1. Launch DControl as Administrator from folder dcontrol
2. Click **"Disable Windows Defender"**
3. Confirm the action

#### Step 3: Run DefenderRemover
1. Launch DefenderRemover
2. Click the **"Remove Defender"** button
3. Wait for completion message
4. currently for now these steps will need to be done after each windows update
